3. Activation Functions
    - Sigmoid
        - Also called call Logistic Activation function / Classification Activation function.
        - Applicable to classification
        - o/p is binary (0 to 1)
        - Shape of sigmoid activavtion function is S Shape
        - It will work on back propagation algorithm
        - mathematical formula is 1/1+e(pow(-x))
        - Sigmoid functions as 2 types
            - Binary Sigmoid function (binary data like 0 to 1), Break point is 0.5, Used for probability
            - Bipolar Sigmoid Functions (out put is -1 to +1 like Movie reviewes), Break point is 0, used for positive or negative
    - Softmax
        - It will comes from multiple sigmoid functions 
        - It will calculate the probability of the output
        - It will find the highest probability
        - Used in Multi-class classification
    - ReLU
        - Rectified Linear Unit
        - Used to make negative values to 0
        - formula is nf(x)=max(0,x)
        - if it is negative value the final value will be rounded to 0
        - if it is a positive value the final value will be x
        - Drawback of ReLU will be Dying ReLU problem 
        - It is making negative values as Zero, even also epochs will be increased and model wont learn anything because neurans got dead.
        - To overcome this function we have another activation function i.e, LeakyReLU 
    - LeakyReLU
        - Instead of making 0, provide small inputs  to the negative values, model will try to learn the inputs again
        - formula f(x)=max(0.1x,x)
        - Instead of small inputs if we provide actial data, we can call Parametric ReLU 
        - ReLU always we need to place with Hidden Layer, should not place with output layers
        - Output layers only contains Sigmoid, Softmax

CNN
    - Convolutional Neural Network
    - Whereever intensity changed drastically automatically Edge will be created.
    - CL(1) will identify the edges (it wont devide image in to pixels) extracting initial edges can called premitive features
    - Collaboration of premitive features is called complex features
    - Filters also called as Kernals, applying filter will tae care by CNN
    - Moving filter to one step to right or one step down technically we cacn called as Stride
    - Because of filteration middle pixels will participate and side pixels will not be participate where we may lose the importent pixels from corners. So we can add dummy data in all corners.
    - Adding dummy data at top and bottom will called Padding
    - CL(n) will identify the full shape
    - Each layer will convert complex layers to premitive layers and Each layer will learn and extract edges
    - CNN overcomes 3 problems
        - High Computations cost
        - Over Fittinig
        - Loss of spacial features
    - Through back propagation algorithm CNN will improve the model accuracy.
    - For every object one feature image will created. Removing empty spaces between 2 images then they will become one image. This type of optimization technique is called Pooling
